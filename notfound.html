Minecraft Status API

To use the server status API, the target server must have enable-query set to true in it's server.properties file.

All requests start with one of the following URLs, and the optional queries can be used to customize/filter the response.

URL Description
https://dags.me​/minecraft​/<server_address> Query the status of the server at the given address, on the default port (25565)
https://dags.me​/minecraft​/<server_address>​/<port> Query the status of the server at the given address, on the given port
Query Type Description
pretty= true/false Specify whether the returned json should be pretty printed (default is false)

Sample URLs

  • https://dags.me/minecraft/example.com?pretty=true
  • https://dags.me/minecraft/example.com/25575?pretty=true

    • Discord Status API

    To use the server status API, the target discord server must have the server widget enabled.

    All requests start with one of the following URLs, and the optional queries can be used to customize/filter the response.

    URL Description
    https://dags.me​/discord​/<server_id> Query the status of the server with the given server id
    Query Type Description
    pretty= true/false Specify whether the returned json should be pretty printed (default is false)

    Sample URLs

  • https://dags.me/discord/207677939146424331?pretty=true
